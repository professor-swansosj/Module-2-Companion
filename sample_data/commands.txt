show lldp neighbors
show inventory

show memory statistics
show clock
show users

# Interface Specific Commands
show interface GigabitEthernet0/0
show interface FastEthernet0/1
show ip interface GigabitEthernet0/0
show interface description
show interface counters
show interface trunk

# Configuration Commands - Use with caution
# These modify device configuration
configure terminal
interface loopback 99
ip address 99.99.99.99 255.255.255.255
description Test Loopback Created by Python
no shutdown
exit
hostname TestRouter
enable secret cisco123
username admin privilege 15 password admin123
line vty 0 4
password cisco
login
exit
ip domain-name lab.local
crypto key generate rsa modulus 1024
ip ssh version 2
end
write memory

show tech-support
show diagnostic result

undebug all
